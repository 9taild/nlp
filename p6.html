#6)Write a program to implement dependency parsing of a given text. #Practical 6 : Dependency Parsing import spacy model = spacy.load('en_core_web_sm') sentence = "The quick brown fox jumps over lazy dog" depend = model(sentence) print("token\tPOS\tDep\tHead") for token in depend: print(f"{token.text}\t{token.pos_}\t{token.dep_}\t{token.head.text}") #####################################
